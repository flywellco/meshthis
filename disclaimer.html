Technical Limitations You Should Know

Smart Decimation MeshThis reduces the triangle count of large scan files to fit within your chosen budget. This is a lossy process — fine surface detail may be reduced. The default 300k triangle budget is appropriate for most hobby prints. Increase the budget for higher-detail output.
Large File Limitation Files above approximately 200MB (Pro) or 80MB (Standard) may return partial geometry or time out due to browser memory limits. WebAssembly heap size varies by browser. Chrome and Edge handle large files best.
Point Cloud vs Mesh Input Some PLY files exported from scan apps contain only point cloud data with no face topology. MeshThis will construct a basic mesh from these points, but the result may not be watertight. For best results, export a "mesh" or "processed" file from your scan app rather than a raw point cloud.
Browser-Dependent Behaviour Conversion results may vary between browsers due to differences in WebAssembly memory limits. We recommend Chrome or Edge on desktop for large files.
